# DIA 2 - Componentes React

## DESCRIÇÃO DOS PROGRAMAS

Aprofundamo-nos nos conceitos de componentes React. A utilização de `props` para passar dados entre os componentes seguindo uma lógica de construção que favorece a dinamização dos componentes. A utilização de `PropTypes` para checar o tipo de uma prop, a presença, o formato do objeto, array de elementos de determinado tipo etc.
<br>
<hr>

Implementar de forma simplificada uma Pokedex. Para os que não estão familiarizados com o universo Pokemon , a Pokedex é uma enciclopédia de todos os pokemons na natureza. Para o seu caso, a sua aplicação precisa mostrar todos os pokemons presentes no arquivo com o objeto `data` fornecido para o exercício.

### Exercício 1

    Criar um componente que represente um pokemon. Esse componente recebe como entrada um objeto que contém informações referentes a um pokemon específico. Esse componente precisa retornar as seguintes informações obrigatórias para serem mostradas para quem usar a aplicação, essas informações devem ser validadas utilizando PropTypes: nome, peso, tipo e imagem do pokemon.


### Exercício 2

    Criar um componente que representa a enciclopédia de pokemons. Esse componente recebe como entrada uma lista de pokemons para serem mostrados na tela.
